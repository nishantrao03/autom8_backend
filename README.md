# autom8_backend

## Description
autom8_backend is a Node.js backend application that sets up a simple Express server. It is designed to handle incoming requests and respond appropriately.

## Project Structure
```
autom8_backend
├── src
│   ├── server.js
│   └── routes
│       └── index.js
├── package.json
└── README.md
```

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd autom8_backend
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Server
To start the server, run the following command:
```
npm start
```
The server will start and listen on port 3000. You can access it at `http://localhost:3000`.

### API Routes
The application includes routes defined in `src/routes/index.js`. You can modify these routes to handle different endpoints as needed.
